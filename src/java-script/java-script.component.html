Type of Variables

In JavaScript, we use var, let, and const to create variables

Var :- If we create a variable outside of a function, we can use it anywhere in code. If we create a variable inside a function, we can only use it within that function. we can re-declare a variable
Let :- The variable is only available inside the block where we define it.Let variables are re-declared in the same scope.
Const:- const is similar to let, but with one big difference: once we assign a value to a const variable, we cant change it later. Like let, const is block-scoped.

There are different type of functions

Anomouis Function:- function which dont have name, the function assigned to a variable

Example:-
let mult = function (x,y)
return x*y
console.log(mult(1,2))

Named function which are commonly used

Example:-
function mult(x,y)
return x*y
console.log(mult(1,2))

Arrow functions also called as fat arrow functions which are directly return the value without return key

Example:-
function mult = (x,y) => x*y;
console.log(mult(1,2))
