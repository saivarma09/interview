What are Injectable?

  • Decorator that marks a class as available to be provided and injected as a dependency.

Directives

  • Directives are used to change the structure style or behaviour of the DOM element
    • Structual Directive:- Ngif, NgFor, NgSwitch
    • Attribute Directive:- Ngclass, NgModel, NgStyle
    • Custom Directive:- @Directive({ standalone: true, selector: '[appHighlight]'})

Service

  • Typescript class used for sharing data or functionality
  • No need to specify behaviour
  • Need to provide in service
  • Need to provide in service
  • Code reuse
  • data sharing between components

Decorators

  • In Angular, decorators are functions that are used to modify the behavior of classes and their members.
  • Decorators are start with @
    • @Input()
    • @Output()
    • @ViewChild()

Lazy loading

  • Lazy loading is a strategy used to optimize the loading time of an Angular application by loading modules only when they are required.

Browser Module

  • It is used to run your application in browser, It includes binding, rendering and event handling

Data Sharing

  • @Input() is used to transfer data from parent to child
  • @Output() is used to transfer data from child to parent
  • @ViewChild() is used to get data from child to parent
  • Subject and BehaviorSubject is used if there is no relation between the components

Local Storage

  • Local storage store data in the client side
  • localStorage.setItem('','') to store the data
  • localStorage.getItem('') to get the stored data
  • localStorage.clear() to clear

Session Storage

  • That allows temporarily store information in the user's browser, session store will clear if browser reload or closed
  • sessionStorage.clear() to clear
  • sessionStorage.setItem('installAppMsg','installed') to store
  • sessionStorage.getItem('') to get stored data

Async Pipe

  • Async pipe is used to get the data from the observal or promise in the HTML.

    • Pipes

  • Pipe are function that transform data before it is displayed to the USer
  • There are two type of pipes

    • Pure

    • Pure pipe will call on params change

    Example: Currency, JSON, LowerCase, UpperCase

    Impure

    • Impure pipe will call on every change
    • By default angular will create pure pipes to impure we need to add pure:false

    Interceptors

    • HttpClient supports a form of middleware known as interceptors.
