RXJS operator are pre define functions. Operators are the essential pieces that allow complex asynchronous code to be easily composed in a declarative manner. There are two type of Operators
  • Creation
    • of
    • from
    • Range
    • interval
  • Pipeable
    • Map
    • SwitchMap
    • DebounceTime
    • Filter
    • Find
    • Reduce
    • Contact
